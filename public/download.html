Developer Mode Required

iOS 16+ requires Developer Mode to install apps outside the App Store. You only need to do this once.

I need to enable Developer Mode
1

Open Settings → Privacy & Security

2

Scroll down and tap Developer Mode → toggle it On

3

Your iPhone will restart — after it reboots, tap Turn On when the prompt appears

4

Come back to this page and continue below

Already enabled? Continue to Step 1 below.
Step 1 — Register Device
1
Tap Register Device — Safari will download and install a configuration profile
2
Open Settings → General → VPN & Device Management and install the profile
3
The profile will show "Profile Installation Failed" or a warning — this is normal and expected. It just means your device has been registered. Tap Done and move on to Step 2.
That error is supposed to happen. The profile registers your device with Apple — the "failed" message is just iOS telling you it couldn't install a management profile, which is fine. Your device is registered. Continue to Step 2.
Step 2 — Install App
After tapping Install App, Safari will show a dialog saying "Open in iTunes" — tap Open to start the install. That dialog is normal on iOS 16+, not an error.

If it says "Unable to Download App", your device isn't in this build yet — wait for a notification that your build is ready.

Open this page on your iPhone in Safari to install the app.
